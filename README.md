## Effects of Temperature Shocks and Desiccation on Intertidal Crabs

This repository has been simplified to a general project template and no longer includes Quarto manuscript scaffolding.

### Repository Structure

- `data/` — project data files
- `scripts/` — analysis and processing scripts
- `results/` — generated outputs and figures
- `Annobibs/` — annotated bibliography materials

### Existing Project Documents

- `Letter of Intent.docx`

### Materials List

- Two tanks with a sectioned off area in each for the green crab
- Thermometer for air/water temperature
- Scale
- Calipers
- 2 heat lamps
- 2 tray corrals
- Mesh to divide corrals to keep crabs separated
- Substrate (mix of sand/rocky) to fill corrals
- Resazurin salt, DI water, DMSO
- 12 cups for resazurin solution
- Two 96-well plates
- Tags to identify crabs
